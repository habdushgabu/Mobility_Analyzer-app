# Streamlit Frontend

This Streamlit app is a dashboard scaffold for the Intelligent Urban Mobility Analytics Platform.

## Launch

```bash
streamlit run frontend/dashboard.py
```

## Tabs
- Dashboard Overview
- Visual Analytics
- SQL Insights
- Spark Analytics
- GenAI Chatbot

## Notes

Add data files to `data/raw/` and run the pipeline scripts before expecting charts and tables to render.
